#　巨戟アーティアのスキル抽選の仕様について

## 巨戟アーティアのスキル抽選について
### グループスキル一つ、シリーズスキル一つずつ抽選される
- グループスキルは[group_skills.yaml](./group_skills.yaml)を参照すること
- シリーズスキルは[series_skills.yaml](./series_skills.yaml)を参照すること

### 激化タイプはテーブルの差に影響しない
- 激化タイプの選択はあるが、抽選テーブルは武器と属性のみで決まる

### 武器の種類は以下の14種類である
- 大剣
- 太刀
- 片手剣
- 双剣
- ハンマー
- 狩猟笛
- ランス
- ガンランス
- スラアク
- チャアク
- 操虫棍
- 弓
- ライトボウガン
- ヘビィボウガン

### 武器に付与される属性は以下の9つである
- 火属性
- 水属性
- 雷属性
- 氷属性
- 龍属性
- 麻痺属性
- 睡眠属性
- 爆破属性
- 毒属性


## 巨戟アーティアのスキル抽選のテーブル仕様について

以下のルールをもとに進行する
- 武器と属性ごとに異なるテーブルが存在する
- テーブルのカーソルは全属性/全武器で共通であり、いずれかのテーブルで変更があると、全テーブルが連動して一つ進む

したがって以下のような方法で効率的なスキルの抽選ができる
1. 武器、属性ごとに10ほどカーソルを進めてテーブルの中身をメモする
2. ゲームを再起動する
3. 1.2.を複数のテーブルに対して実行する
4. 同一のカーソル内で最も結果の良い武器、属性テーブルを選んでスキル抽選を行う
