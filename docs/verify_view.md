# 確認画面 仕様

## 目的
保存済みの抽選結果を一覧で確認できるようにする。

## 表示内容
- 行: 同一カーソルの結果を1行にまとめる
- 列:
  - カーソル: `cursorId`
  - テーブル: `武器::属性` のパターンごとに1列
    - 各セルに「お気に入り/シリーズ/グループ」をブロック表示する

## 並び順
- カーソル昇順
- テーブル列は `武器/属性` の昇順

## 表示ルール
- テーブル名は `武器::属性` をそのまま表示
- お気に入りは「お気に入り/通常」として表示する

## 画面導線
- 上部の固定タブに「確認」を追加し、他の画面と切り替えられるようにする

## エクスポート/インポート
- ローカルストレージの内容をJSONとしてエクスポートできる
- JSONファイルをインポートしてローカルストレージへ復元できる
