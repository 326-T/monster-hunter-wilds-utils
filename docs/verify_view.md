# 確認画面 仕様

## 目的
保存済みの抽選結果を一覧で確認できるようにする。

## 表示内容
- 行: 抽選結果1件につき1行
- 列:
  - カーソル: `cursorId`
  - テーブル: `武器/属性/激化タイプ` を `/` 区切りで連結した文字列
- シリーズ
- グループ
  - お気に入り
  - 登録日時

## 並び順
- カーソル昇順
- 同一カーソル内ではテーブル名の昇順

## 表示ルール
- テーブル名は `武器/属性/激化タイプ` をそのまま表示
- お気に入りはチェック表示のみ

## 画面導線
- 上部の固定タブに「確認」を追加し、他の画面と切り替えられるようにする

## エクスポート/インポート
- ローカルストレージの内容をJSONとしてエクスポートできる
- JSONファイルをインポートしてローカルストレージへ復元できる
