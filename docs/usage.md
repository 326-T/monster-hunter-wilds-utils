# 使い方

このサイトが提供する機能は2つのみです。UXに絞って説明します。

## 1. 各テーブルの抽選結果を保存する画面
- 武器×属性×激化タイプごとのリンク一覧からテーブルへ移動する
- テーブル画面で、グループスキル/シリーズスキルをフォーム入力して行を追加する
- 気に入った抽選結果はお気に入りにできる
- お気に入りは行にチェックが付くのみで、並び順は常に `created_at` の昇順
- カーソルが通過済みの行はデフォルト非表示で、表示ボタンで切り替えられる
- お気に入りは次の画面で優先表示される

## 2. カーソルを進める画面
- カーソルは全属性/全武器/全激化タイプで共通
- 武器/属性/激化タイプでフィルターできる
- フィルター後、そのカーソルで取得できる抽選結果のうち NULL でないものをすべて表示する
- この一覧ではお気に入りを優先表示する
- その中から1つを選ぶとカーソルが進み、次の候補が表示される
- カーソルが進むとき、カーソル位置がまだ NULL のテーブルは
  シリーズスキル/グループスキルともに「不明」と表示する
- 「不明」は `docs/group_skills.yaml` / `docs/series_skills.yaml` の末尾にある選択肢で、
  通常の選択肢と同じ扱いにする

## NULL の定義
ローカルストレージに保存されたテーブルごとのJSONデータで、
カーソルに該当するIDを持つレコードが存在しない場合を NULL とする。
